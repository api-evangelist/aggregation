About API Aggregation

This is a project site to track on the different approaches to API aggregation.

I host individual research projects and sections of API Evangelist as their own Github repositories. This way projects can evolve on their, independent of the overall site.

Aggregation represents an important trend in the evolution of APIs, and the need to mix, reuse and evolve API resources..

If you have any questions about this research, contact me at info@apievangelist.com.
