API Provides 3rd Party and In-House Developers With Fast and Cost-Effective Integration Into Leading Cloud Messaging Services DENVER, CO, Jul 22, 2014 (Marketwired via COMTEX) — Cloud API integration and aggregation company, Cloud Elements, today announced the availability of Infobip through their Messaging Hub API.Infobip, a global provider of mobile solutions connecting mobile network operators and enterprises through an in-house developed and operated mobile services cloud, will join leading cloud email infrastructure service, SendGrid, in offering international 1-way and 2-way SMS capabilities as a part of the Cloud Elements Messaging Hub.These popular cloud messaging services can be integrated via a single uniform application programming interface (API), allowing developers to easily access and leverage all Infobip and SendGrid services.Cloud Elements customers will now be able to easily add international SMS capabilities to their application’s email proposition.Software developers will use the same integration to send emails via SendGrid as they do to send SMS texts via Infobip — seamlessly adding another channel for their application to deliver communications to staff, customers or stakeholders.
