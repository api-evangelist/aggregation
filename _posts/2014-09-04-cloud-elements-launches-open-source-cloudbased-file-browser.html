DENVER, CO–(Marketwired – Sep 4, 2014) – Leading cloud API integration and aggregation service, Cloud Elements, today announced the launch of Filebrowser.io, a free, open source, cloud file browser.The JavaScript library connects to the leading cloud document services including Box, Dropbox, Google Drive and Microsoft OneDrive.Developers can integrate the open source library into their application, allowing their customers to access cloud storage services without leaving the application.Filebrowser.
